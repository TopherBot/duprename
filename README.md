# duprename – Auto‑rename duplicate files

**duprename** is a single‑file Python command‑line tool that detects files with identical (case‑insensitive) names in a given folder and renames the later ones by appending a numeric suffix.

## Features
- Scans a target directory (non‑recursive by default).
- Detects duplicate names regardless of case (`photo.jpg` vs `PHOTO.JPG`).
- Renames duplicates as `name (1).ext`, `name (2).ext`, …
- Dry‑run mode to preview changes.
- Zero‑dependency, runs on any Python 3.8+.

## Installation
```bash
# Option 1: pipx (recommended)
python -m pip install --user pipx
pipx install git+https://github.com/youruser/duprename.git

# Option 2: direct script copy
curl -L https://raw.githubusercontent.com/youruser/duprename/main/duprename.py -o duprename.py
chmod +x duprename.py
```

## Usage
```bash
# Rename duplicates in the current directory
duprename .

# Rename duplicates in a specific folder
duprename /path/to/folder

# Show what would happen without renaming
duprename --dry-run .
```

## How it works
1. The script walks the target directory (non‑recursive) and builds a map of lower‑cased filenames to their original paths.
2. If a name is seen more than once, a new name is generated by appending ` (n)` before the file extension, where `n` increments until the name is unique.
3. Files are renamed using `os.rename`.
